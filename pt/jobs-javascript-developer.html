Vaga para Javascript Developer

Você precisa dominar Javascript, Git, ter bom conhecimento de HTML, CSS e UX web.

Aqui não existem diferentes níveis de programadores. Temos times eficientes porque prezamos por programadores de alto nível, que são capazes de se comunicar bem e trabalhar diretamente com os clientes.

Em todos os projetos trabalhamos com entregas semanais de software funcionando. Uma iteração começa com a Reunião de Planejamento com o cliente e termina com a entrega do Roteiro de Testes, de acordo com a nossa metodologia.

Todos os membros do time, incluindo os programadores, são responsáveis pela prototipação e conceito das telas do projeto. Trabalhamos com desenhos feitos a mão, em folhas A4, como forma de comunicação sobre o trabalho que deve ser realizado. Isso quer dizer que todos são responsáveis pela Experiência do Usuário e Usabilidade das aplicações.

É necessário ter noções de inglês: leitura, escrita e oral. Falar fluentemente é um diferencial. Temos clientes em todos os cantos do mundo e a língua padrão de comunicação dentro do código é o inglês.

Temos um escritório super irado no Rio de Janeiro, mas você poderá trabalhar remotamente do seu home office, algum café na sua cidade ou mesmo mochilando pela Europa.

Cada um pode escolher as ferramentas com que prefere trabalhar. Gostamos de Macs. Fornecemos um para cada designer do time.

Aqui o que importa é saber se auto-gerir, trabalhar de forma altamente colaborativa em time e criar produtos fantásticos para os clientes!

Quer conhecer mais sobre a nossa cultura? Leia em sobre nós.

O que esperamos de um Javascript Developer?

  • Ter conhecimento nos frameworks atuais como Angular, Ember, React.js e etc..
  • Ter sólido conhecimento em Javascript e conseguir escrever testes para validar o código
  • Ter experiência com projetos usando Javascript em produção
