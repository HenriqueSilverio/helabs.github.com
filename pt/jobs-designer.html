Vaga para Web Designer

Nossos designers trabalham nos produtos web de nossos clientes. Você precisa saber direcionar o projeto para sua implementação mais simples e que vai permitir que o projeto seja validado o mais rapidamente possível no mercado. Isso vai garantir que estaremos fazendo o software certo. Você irá trabalhar diretamente com os clientes e usuários e por isso precisa ter boa comunicação verbal e escrita.

O trabalho a ser realizado nos projetos é priorizado pelo cliente, mas independentemente disso, esperamos que você crie interfaces que sejam bonitas e funcionais.

Os designers trabalham aplicando os layouts criados por eles próprios diretamente no código dos projetos. Conhecimentos de HTML e CSS são necessários. O trabalho de codificação do designer é essencial para poder existir implementação rápida de modificações na interface, o que resulta num ciclo curto entre hipótese e validação da usabilidade de uma interface com um usuário.

Cada um pode escolher as ferramentas com que prefere trabalhar. Gostamos de Macs. Fornecemos um para cada designer do time.

É necessário falar inglês fluentemente: leitura, escrita e oral. Temos clientes em todos os cantos do mundo e a língua padrão de comunicação dentro da empresa é o inglês.

Nossas iterações de trabalho são de uma semana, de acordo com a nossa metodologia.
